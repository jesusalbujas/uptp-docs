# SQL: El Lenguaje de los Datos

Qué es SQL y lo que debes saber ahora mismo.

## ¿Qué es SQL?
SQL (Structured Query Language) es el lenguaje estándar para gestionar bases de datos relacionales.

## Comandos Fundamentales
- `SELECT`: Recuperar datos.
- `INSERT`: Agregar nuevos datos.
- `UPDATE`: Modificar datos existentes.
- `DELETE`: Eliminar datos.
