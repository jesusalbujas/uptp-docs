# Docker: Contenedores para Todos

Descubre los contenedores Docker y cómo trabajar con él.

## ¿Qué es Docker?
Docker es una plataforma abierta para desarrollar, enviar y ejecutar aplicaciones. Docker permite separar las aplicaciones de la infraestructura, entregando software rápidamente.

## Conceptos Clave
- **Imagen**: Plantilla de lectura única con instrucciones para crear un contenedor.
- **Contenedor**: Instancia ejecutable de una imagen.
- **Dockerfile**: Archivo de texto con los comandos para construir una imagen.
