# Ruta de Aprendizaje para el Ingeniero en Informática

Bienvenido a la carrera de Ingeniería en Informática. Esta guía te orientará sobre qué tecnologías y conceptos fundamentales deberías empezar a explorar.

## Fundamentos
- Lógica de Programación
- Algoritmos y Estructuras de Datos
- Arquitectura de Computadoras

## Desarrollo Web
- HTML5, CSS3, JavaScript
- Frameworks modernos (React, Vue, Angular)

## Backend y Bases de Datos
- Node.js, Python, Java
- SQL (PostgreSQL, MySQL) vs NoSQL (MongoDB)

## Herramientas Esenciales
- Git y GitHub
- Terminal Linux
