# Trayectos II, III y IV: Proyectos de Software

Estos proyectos comparten una estructura orientada al **Desarrollo de Software**. Los temas son similares, pero la profundidad de la documentación y los entregables se incrementan en cada Trayecto.

## Estructura Común
1. Planificación y Diagnóstico
2. Requerimientos, Diseño y Desarrollo
3. Implantación y Pruebas
4. Resultados y Logros
5. Manuales
